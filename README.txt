Para a extração das imagens, utilize readimages.py O nome dos arquivos que contêm as images e os labels será pedido na execução.
Também será pedido o nome dos diretórios:
Para o diretório dos dados de treinamento, utilize "dataset" sem aspas
Para o diretório dos dados de teste, utilize "testdata" sem aspas
É necessário criar o diretório antes de rodar o programa

O programa principal é o run.py
featureextraction.py contêm as funções utilizadas para extrair as features das imagens

Caso utilize o jupyter notebook, pode-se utilizar o run.ipynb no lugar do run.py

O código que produz os gráficos estão comentados no arquivo run.py

Verifique a versão do opencv caso dê erro na linha 147, versão utilizada foi a 4.0
